# Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn, customer behavior, revenue performance, and customer engagement using Power BI.

## Tools Used

* Power BI
* DAX
* Data Analysis
* Data Visualization

## Dataset

The dataset contains customer demographics, purchasing behavior, loyalty information, referral sources, and churn indicators.

## Key Metrics

* Total Customers: 1,000
* Total Orders: 65,000
* Total Revenue: $6.36 Million

## Dashboard Features

* Customer Churn Analysis
* Revenue Analysis by Country
* Revenue by Referral Source
* Device Usage Analysis
* Customer Segmentation
* Age Group Analysis
* Loyalty Tier Analysis
